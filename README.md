# Muslim Student Jama'a Portal

![Project Status](https://img.shields.io/badge/status-active-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue)

## Project Overview

The Muslim Student Jama'a Portal is a comprehensive web platform designed to serve as a central hub for Muslim students across Ethiopia. This portal facilitates community building, knowledge sharing, and spiritual growth by providing essential resources and services tailored to the needs of Muslim students.

## Key Features

### Multi-Language Support
- Available in English, Amharic, Afaan Oromo, and Arabic
- Seamless language switching capability

### Core Functionalities
- **Community Resources**: Access to prayer times, Islamic events, and community announcements
- **Educational Hub**: Digital library with Islamic texts, academic resources, and study materials
- **Learning Opportunities**: Halaqat (study circle) schedules and registration
- **Community Engagement**: Sector registration for various volunteer opportunities
- **Membership System**: Student registration and profile management

### Interactive Components
- Dynamic prayer times display
- Daily Azkar (remembrance) tracking
- Event calendar with upcoming activities
- Community photo gallery
- Contact and communication channels

## Technical Implementation

This project is built as a static website using vanilla web technologies:

- **HTML5**: Semantic markup and structure
- **CSS3**: Responsive design with modern aesthetics
- **JavaScript**: Client-side interactivity and dynamic content
- **Lucide Icons**: Consistent iconography throughout the interface

The portal follows a single-page application (SPA) architecture with client-side routing for seamless navigation between different sections.

## Sections

1. **Home**: Dashboard with prayer times, upcoming events, and quick links
2. **About**: Mission, vision, and objectives of the Jama'a
3. **Library**: Digital repository of Islamic and academic resources
4. **Halaqat**: Study circle schedules and registration
5. **Events**: Community events and prayer times
6. **Activities**: Student community activities and programs
7. **Sectors**: Volunteer sector registration (Da'awa, Academic, Information, Charity, Social)
8. **Contact**: Communication channels and location information
9. **Registration**: Student membership registration

## Getting Started

### Prerequisites

To run this project, you only need a modern web browser. No additional software installation is required.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/muslim-student-jamaa-portal.git
   ```

2. Navigate to the project directory:
   ```bash
   cd muslim-student-jamaa-portal
   ```

3. Open `index.html` in your preferred web browser.

Alternatively, you can simply double-click the `index.html` file to open it in your default browser.

## Usage

1. Open the website in your browser
2. Navigate through different sections using the menu
3. Switch languages using the language selector in the navigation bar
4. Register as a member using the "Join" button
5. Explore resources in the Library section
6. Join study circles (Halaqat) through the registration process
7. Participate in community activities and volunteer for sectors
